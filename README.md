# Adventure Game – Python Text-Based RPG

This is a text-based adventure game written in **Python**. Players explore different rooms, interact with NPCs, collect items, and battle enemies. The game demonstrates object-oriented programming concepts, file handling, and simple game logic.

---

## Features

- **Player Character**: Move between rooms, pick up items, track inventory and health.
- **Rooms & Map**: Explore different rooms like Welcome Room, Main Hallway, Treasure Room, Greenhouse, and Potion Room.
- **Enemies**: Face enemies such as Zombies and Witches with turn-based combat.
- **NPCs**: Interact with friendly NPCs who provide tips or story elements.
- **Items**: Collect items like keys, gold, rubies, potions, and healing herbs.
- **Save/Load System**: Save the current game state to a JSON file and load it later.
- **Combat System**: Turn-based attacks with random damage values.
- **Status Display**: View player’s health, inventory, and location at any time.

---

## Technologies

- Python 3
- Object-Oriented Programming (Classes for Player, NPC, Enemy)
- JSON for saving/loading game state
